# 抽卡分析器 / Wuthering Waves Gacha Analyzer

一個抽卡數據追蹤與分析工具，開發中。


## 🚀 快速開始

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

開啟瀏覽器訪問 `http://localhost:5173`

### 打包生產版本

```bash
npm run build
```

打包後的文件會在 `dist/` 目錄下。

## 📦 技術棧

- **React** - UI 框架
- **Vite** - 構建工具
- **Tailwind CSS v4** - 樣式框架
- **lucide-react** - 圖標庫

## 🎮 如何使用

1. 選擇遊戲類型（鳴潮 / 原神 / 星鐵）
2. 按照指引在遊戲中執行 PowerShell 指令獲取抽卡 URL
3. 貼上 URL 並匯入資料
4. 查看你的抽卡統計和歐非排名！
